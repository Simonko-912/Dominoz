# Contributing to Dominoz

Thanks for contributing! By participating in this project you will agree to follow following rules and contribute.

## How to contribute? 

1. Report bugs
 - Report bugs and fix bugs, with reporting bugs with errors, info , steps to reproduce etc.

2. Suggest features
 - Suggest features in issues, include examples how it works

3. Submit Pull Requests (PRs)
 - Fork the repo and create a new branch for your changes:  
     ```bash
     git checkout -b feature/my-feature
     ```  
 - Make your changes, commit with a clear message:  
     ```bash
     git commit -m "Add feature X"
     ```  
 - Push your branch to your fork:  
     ```bash
     git push origin feature/my-feature
     ```  
 - Open a pull request against the `main` branch in the original repo.

4. Test code
 - Before submiting PRs make sure your code doesnt have bugs or doesnt run.
